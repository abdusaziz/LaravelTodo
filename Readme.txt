1. Open project location
2. >git clone https://github.com/abdusaziz/LaravelTodo.git
3. >cd LaravelTodo
4. >code . to open VisualStudio Code
5. >composer update
6. Duplicte .env.example file & rename to .env
7. in .env file update DB_DATABASE=laraveltodoapp
8. make a database in mysql server, named "laraveltodoapp"
9. >php artisan key:generate
10. >npm install
11. >npm run dev
12. >php artisan serve to test and its running well
13. >php artisan migrate

The app is ready to use.
